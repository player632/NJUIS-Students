# 线性代数

## 基本信息

课程名称：线性代数

课程介绍：线性代数是一门以**向量、矩阵、线性方程组**为核心研究对象，探究线性空间结构、线性变换规律及相关数值计算方法的数学分支。其搭建起连接纯数学与工程应用的桥梁，不仅是解决多元线性方程组、数据降维等问题的关键工具，也是学习机器学习、信号处理等前沿领域的必备前置知识。

课程画像：

- 课程类型：通修
- 学分：4 学分
- 考核方式：平时 20% · 期中考试 30% · 期末考试 50%

## 经验分享区

课程定位评价： 

线性代数的入门难度不算高，但想要吃透向量空间、线性变换这些抽象概念，以及熟练掌握矩阵对角化、特征值求解等复杂运算，还是需要投入不少时间去理解和练习；另外，有关线性代数的证明题往往会比较困难，有时候可以结合线性代数的本质去理解和分析。因此对于大一新生而言，这门课整体**难度较高**。

考试情报：

- 题型整体分布：期中、期末考试题型整体较为稳定，以计算题和证明题为主，题量适中。
- 常规考点：计算题仍以课后习题、往年常规题（如矩阵求幂、施密特正交化、具体二次型化标准形）的直接或变形为主，需要极高的计算熟练度和准确性。
- **⚠️ 重大变动**：2025-2026第一学期，有消息称命题组引入了“匿名专家”:sob:。该学期难度骤升，特别是**显著加强了对“抽象空间”和“逻辑构造”的考察**。
- 难度波动规律：结合近几年的情况看，秋季学期的线代难度通常会略高于春季学期。智科专业学生在1-1学期修读，请务必做好一定的心理建设和难度预案。
- 对计算能力的要求：部分题目计算量极大且步骤繁琐，对计算准确性、速度及步骤规范性要求极高。一旦中间过程算错，极易导致整题崩盘。

### 高分策略:

- **吃透作业与例题，但不止于刷题**：必须搞懂每一道作业题和教材例题的“为什么”。建议在复习时制作一份“**思维导图**”，但不要只列公式，必须串联起概念之间的转化关系。

- **刷往年试卷十分重要**：
    - 建议考前一定要独立完整做多套往年期中/期末试卷，熟悉常见题型；时间充裕的情况下也可以做未布置的书后习题，题目质量也很高。

- **证明题突破**：
    - 鉴于匿名专家偏爱“教材定理证明”，复习时建议别跳过教材中的黑体字定理证明。
    - 平时做题时，遇到证明题不要只求看懂答案，合上书本，拿白纸自己推一遍。

- :warning:遇到完全没思路的抽象证明或构造题，**绝对不要空着**！可以默写题目涉及的核心定义，或是写出相关的通用公式。这些步骤分在高难度题目中往往能 ~~捞捞~~ 救你一命。

- **其他提醒**：
    - 如果遇到“匿名专家”返场，一定要**稳住心态**————大部分人都会被难住，把会做的计算题全做对，较难的证明题试着写写思路，你就赢了:relaxed:
